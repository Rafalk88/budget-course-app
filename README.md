## Run Locally

Clone the project

```bash
  git clone https://github.com/Rafalk88/budget-course-app
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
  yarn install
```

Start the server and json-server with one command

```bash
  npm run dev
  yarn dev
```
